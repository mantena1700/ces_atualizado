# Componentes Gráficos Base (UI Kit)

Coleção de componentes atômicos construídos sobre Tailwind CSS e Radix UI, garantindo consistência visual e acessibilidade.

## Componentes Disponíveis:

- **`dialog.tsx`**: Novo sistema de modais (Dialog) utilizando primitivos do Radix UI. Implementa animações de entrada (FadeIn/ZoomIn), backdrop blur e foco acessível.
- **`button.tsx`**: Botão polimórfico com diversas variantes (default, outline, ghost, link, destructive).
- **`card.tsx`**: Container base com suporte a cabeçalho, conteúdo e rodapé, seguindo o padrão visual de "Cards Flutuantes".
- **`input.tsx`**: Campo de entrada de texto estilizado, com estados de foco e erro.
- **`label.tsx`**: Rótulo de formulário acessível.
- **`badge.tsx`**: Chip/Tag para status e categorização.
- **`table.tsx`**: Estrutura responsiva para tabelas de dados.
- **`tabs.tsx`**: Navegação por abas.
- **`progress.tsx`**: Barra de progresso (Radix UI) ✨ NOVO
  - Prop `indicatorClassName` para customizar cor do indicador
  - Usado em dashboards de orçamento para mostrar utilização
